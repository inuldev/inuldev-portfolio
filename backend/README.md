## Environment Setup

1. Copy config.example.env untuk membuat konfigurasi development:

```bash
cp config/config.example.env config/config.development.env
```

2. Copy config.example.env untuk membuat konfigurasi production:

```bash
cp config/config.example.env config/config.production.env
```

3. Update nilai-nilai konfigurasi sesuai environment

## Menjalankan Aplikasi (Development dan Production)

Development mode:

```bash
npm run dev
```

Production mode:

```bash
npm run prod
```
